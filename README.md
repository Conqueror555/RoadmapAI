# RoadmapAI

RoadmapAI is an AI-powered career roadmap generator built using React, TypeScript, Supabase, and Gemini API.

Features:
- Career roadmap generator
- Skill learning roadmap
- Course roadmap
- Project roadmap
- Authentication with Supabase
- Interactive roadmap viewer

Tech Stack:
- React + TypeScript
- Tailwind CSS
- Supabase
- Gemini API
- React Flow
